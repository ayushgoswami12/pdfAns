Setting up virtual environment 
uv venv 
copy paste soemthing like  ".venv\Scripts\activate"

you have to ask ai to create an requirement.txt which includes all the dependencies 
After you have pasted all the dependencies you have to do : 
uv pip install -r .\requirements.txt
That will install all the dependencies that are present in the ' requirement.txt '
